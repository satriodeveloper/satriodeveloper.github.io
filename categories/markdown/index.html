Mempelajari Perintah Dasar Markdown

Pada artikel ini, kita akan mempelajari perintah dasar markdown. Markdown adalah sebuah bahasa markup yang digunakan untuk menulis sebuah artikel. Markdown memiliki banyak sekali perintah yang dapat digunakan untuk menulis sebuah artikel. Pada artikel ini, kita akan mempelajari perintah dasar markdown. Pada beberapa CMS seperti WordPress, kita terbiasa menulis menggunakan text editor. Namun, pada markdown kita tidak perlu menggunakan text editor untuk menulis sebuah artikel.
