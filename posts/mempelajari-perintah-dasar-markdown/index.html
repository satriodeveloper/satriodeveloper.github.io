Mempelajari Perintah Dasar Markdown

November 5, 2022

Pada artikel ini, kita akan mempelajari perintah dasar markdown. Markdown adalah sebuah bahasa markup yang digunakan untuk menulis sebuah artikel. Markdown memiliki banyak sekali perintah yang dapat digunakan untuk menulis sebuah artikel. Pada artikel ini, kita akan mempelajari perintah dasar markdown.

Pada beberapa CMS seperti WordPress, kita terbiasa menulis menggunakan text editor. Namun, pada markdown kita tidak perlu menggunakan text editor untuk menulis sebuah artikel. Kita dapat menulis sebuah artikel dengan menggunakan markdown. Markdown memiliki banyak sekali perintah yang dapat digunakan untuk menulis sebuah artikel. Pada artikel ini, kita akan mempelajari perintah dasar markdown.

Perintah Dasar Markdown#

Berikut adalah perintah dasar markdown yang dapat digunakan untuk menulis sebuah artikel.

Judul#

Untuk membuat judul, kita dapat menggunakan tanda pagar (#). Semakin banyak tanda pagar yang digunakan, maka semakin kecil ukuran judulnya. Berikut adalah contoh penggunaan tanda pagar (#) untuk membuat judul.

Judul 1#

Judul 2#

Judul 3#

Judul 4#

Judul 5#
Judul 6#

Paragraf#

Untuk membuat paragraf, kita dapat menggunakan tanda enter. Berikut adalah contoh penggunaan tanda enter untuk membuat paragraf.

Paragraf 1

Paragraf 2

Bold#

Untuk membuat teks bold, kita dapat menggunakan tanda bintang dua (**) pada awal dan akhir teks. Berikut adalah contoh penggunaan tanda bintang dua (**) untuk membuat teks bold.

**Teks bold**

Italic#

Untuk membuat teks italic, kita dapat menggunakan tanda bintang satu () pada awal dan akhir teks. Berikut adalah contoh penggunaan tanda bintang satu () untuk membuat teks italic.

*Teks italic*

Strikethrough#

Untuk membuat teks strikethrough, kita dapat menggunakan tanda tanya dua (__) pada awal dan akhir teks. Berikut adalah contoh penggunaan tanda tanya dua (__) untuk membuat teks strikethrough.

__Teks strikethrough__

Link#

Untuk membuat link, kita dapat menggunakan tanda kurung siku ([]) pada awal dan akhir teks. Berikut adalah contoh penggunaan tanda kurung siku ([]) untuk membuat link.

[Teks link](https://www.google.com "Google")

Gambar#

Untuk menampilkan gambar, kita dapat menggunakan tanda kurung siku ([]) pada awal dan akhir teks. Berikut adalah contoh penggunaan tanda kurung siku ([]) untuk menampilkan gambar.

![Teks gambar](https://www.google.com/images/googlelogo_color_272x92dp.png "Google")

Daftar#

Untuk membuat daftar, kita dapat menggunakan tanda minus (-) atau tanda bintang (*) pada awal teks. Berikut adalah contoh penggunaan tanda minus (-) atau tanda bintang (*) untuk membuat daftar.

- Daftar 1

* Daftar 1

Daftar Terurut#

Untuk membuat daftar terurut, kita dapat menggunakan tanda nomor pada awal teks. Berikut adalah contoh penggunaan tanda nomor untuk membuat daftar terurut.

1. Daftar 1

Tabel#

Untuk membuat tabel, kita dapat menggunakan tanda garis atau pipe (|) untuk memisahkan antar kolom. Berikut adalah contoh penggunaan tanda garis atau pipe (|) untuk membuat tabel.

| Name  | Age |
| ----- | --- |
| Bob   | 27  |
| Alice | 23  |

Kode#

Untuk menampilkan kode, kita dapat menggunakan tanda backtick pada awal dan akhir teks. Berikut adalah contoh penggunaan tanda backtick untuk menampilkan kode.

`Teks kode`
