Mempelajari Perintah Dasar Hugo

November 5, 2022

Pada artikel kali ini saya akan mempelajari perintah dasar Hugo. Perintah dasar Hugo ini akan membantu kita untuk membuat konten, mengelola konten, dan mengelola situs web Hugo. Berbeda dengan kebanyakan CMS, Hugo tidak memiliki antarmuka pengguna. Semua perintah Hugo dijalankan melalui terminal. Jadi, sebelum mempelajari perintah dasar Hugo, Anda harus menginstal Hugo terlebih dahulu.

Instalasi Hugo#

Pada artikel ini saya menggunakan sistem operasi Ubuntu 22.04. Jadi, untuk menginstal Hugo pada sistem operasi Ubuntu 22.04, Anda dapat menjalankan perintah berikut pada terminal Anda.

$ sudo apt-get install hugo

atau jika menggunakan brew pada macOS, Anda dapat menjalankan perintah berikut pada terminal Anda.

$ brew install hugo

Setelah Hugo terinstal, Anda dapat memeriksa versi Hugo yang terinstal dengan menjalankan perintah berikut pada terminal Anda.

$ hugo version

Maka saatnya Anda bisa menginstal Hugo pada sistem operasi Anda dengan menggunakan perintah sebagai berikut.

$ hugo new site mysite

Pada perintah diatas saya menggunakan nama situs web Hugo saya adalah mysite. Jika Anda ingin menggunakan nama situs web Hugo yang berbeda, Anda dapat menggantinya dengan nama situs web Hugo yang Anda inginkan.

Membuat Konten#

Setelah Anda menginstal Hugo, Anda dapat membuat konten dengan menggunakan perintah berikut.

$ hugo new posts/mempelajari-perintah-dasar-hugo.md

Pada perintah diatas saya menggunakan nama konten Hugo saya adalah mempelajari-perintah-dasar-hugo.md. Jika Anda ingin menggunakan nama konten Hugo yang berbeda, Anda dapat menggantinya dengan nama konten Hugo yang Anda inginkan.

Mengelola Konten#

Setelah Anda membuat konten, Anda dapat mengelola konten dengan cara mengedit konten tersebut. Untuk mengedit konten, Saya cenderung menggunakan VS Code. Anda dapat mengedit konten dengan menggunakan editor yang Anda sukai. Jika Anda menggunakan VS Code, Anda dapat mengedit konten dengan menjalankan perintah berikut pada terminal Anda.

$ code content/posts/mempelajari-perintah-dasar-hugo.md

Mengelola Situs Web Hugo#

Setelah Anda membuat konten, Anda dapat mengelola situs web Hugo dengan menjalankan perintah berikut pada terminal Anda.

$ hugo server -D

Maka Hugo akan menjalankan server Hugo pada alamat http://localhost:1313. Jika Anda ingin mengubah port server Hugo, Anda dapat mengubahnya dengan menambahkan parameter -p pada perintah diatas.

$ hugo server -D -p 8080

Maka Hugo akan menjalankan server Hugo pada alamat http://localhost:8080. Jika Anda ingin mengubah alamat server Hugo, Anda dapat mengubahnya dengan menambahkan parameter -b pada perintah diatas.

$ hugo server -D -b http://localhost:8080

Maka Hugo akan menjalankan server Hugo pada alamat http://localhost:8080.

Membuat Situs Web Hugo#

Setelah Anda mengelola situs web Hugo, Anda dapat membuat situs web Hugo dengan menjalankan perintah berikut pada terminal Anda.

$ hugo

Maka Hugo akan membuat situs web Hugo pada direktori public. Jika Anda ingin mengubah direktori situs web Hugo, Anda dapat mengubahnya dengan menambahkan parameter -d pada perintah diatas.

$ hugo -d docs

Maka Hugo akan membuat situs web Hugo pada direktori docs.

Membuat Situs Web Hugo Statik#

Setelah Anda membuat situs web Hugo, Anda dapat membuat situs web Hugo statik dengan menjalankan perintah berikut pada terminal Anda.

$ hugo --minify

Maka Hugo akan membuat situs web Hugo statik pada direktori public. Jika Anda ingin mengubah direktori situs web Hugo statik, Anda dapat mengubahnya dengan menambahkan parameter -d pada perintah diatas.

$ hugo --minify -d docs

Maka Hugo akan membuat situs web Hugo statik pada direktori docs.

Membuat Situs Web Hugo Statik dengan CDN#

Setelah Anda membuat situs web Hugo statik, Anda dapat membuat situs web Hugo statik dengan CDN dengan menjalankan perintah berikut pada terminal Anda.

$ hugo --minify --baseURL https://example.com

Maka Hugo akan membuat situs web Hugo statik dengan CDN pada direktori public. Jika Anda ingin mengubah direktori situs web Hugo statik dengan CDN, Anda dapat mengubahnya dengan menambahkan parameter -d pada perintah diatas.

$ hugo --minify --baseURL https://example.com -d docs

Maka Hugo akan membuat situs web Hugo statik dengan CDN pada direktori docs.

Membuat Situs Web Hugo Statik dengan CDN dan Subdirektori#

Setelah Anda membuat situs web Hugo statik dengan CDN, Anda dapat membuat situs web Hugo statik dengan CDN dan subdirektori dengan menjalankan perintah berikut pada terminal Anda.

$ hugo --minify --baseURL https://example.com/blog

Maka Hugo akan membuat situs web Hugo statik dengan CDN dan subdirektori pada direktori public. Jika Anda ingin mengubah direktori situs web Hugo statik dengan CDN dan subdirektori, Anda dapat mengubahnya dengan menambahkan parameter -d pada perintah diatas.

$ hugo --minify --baseURL https://example.com/blog -d docs

Maka Hugo akan membuat situs web Hugo statik dengan CDN dan subdirektori pada direktori docs. Demi keamanan, Anda dapat mengubah baseURL dengan alamat situs web Hugo Anda.
